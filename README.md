✍️ MERN Blog Website
Welcome to the MERN Blog Website repository! This is a fully functional, full-stack blogging platform built using the MERN stack.

✨ Key Features
Secure Authentication: User login and registration using JWT.

Full Blog Management: Easily Create, Read, Update, and Delete (CRUD) blog posts.

Interactive Content: Includes a Rich Text Editor, Image Support, and a dynamic Comment System.

Search & Discover: Quickly find posts using keywords and categories.

Responsive Design: Optimized for all screen sizes (mobile, tablet, desktop).

⚙️ Technologies
This project utilizes the MERN stack and modern development tools.
Frontend : React.js, Redux, React Router, TailwindCSS
Backend : Node.js, Express.js, JWT (for Auth)
Database : MongoDB (via Mongoose)
